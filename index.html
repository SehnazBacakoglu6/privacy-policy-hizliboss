Gizlilik Politikası

Bu gizlilik politikası, HızlıBoss uygulamasının kullanıcı verilerini nasıl topladığını, kullandığını, sakladığını ve koruduğunu açıklar...

Giriş Bu gizlilik politikası, HızlıBoss uygulamasının kullanıcı verilerini nasıl topladığını, kullandığını, sakladığını ve koruduğunu açıklar. Uygulamamızı kullanarak bu politikada belirtilen şartları kabul etmiş olursunuz. 1. Toplanan Bilgiler Uygulamamızda kullanıcıların kayıt olabilmesi ve sisteme giriş yapabilmesi için aşağıdaki bilgiler toplanır: E-posta adresi (kayıt sırasında) Firma Id bilgisi API kullanıcı adı ve API şifresi (firma verilerine erişim amacıyla) Uygulama kullanımına dair teknik veriler 2. Bilgilerin Kullanım Amacı Toplanan bilgiler şu amaçlarla kullanılır: Kullanıcı hesabının oluşturulması ve doğrulanması Firma verilerine API üzerinden erişim sağlanması Uygulamanın güvenliğini ve sürekliliğini sağlamak Destek ve iletişim hizmetleri sunmak 3. Verilerin Saklanması Kullanıcı verileri güvenli bir şekilde Firebase altyapısında saklanır. API kullanıcı adı ve şifreleri şifrelenmiş (encrypted) olarak tutulur. Veriler yalnızca uygulamanın çalışması için gerekli süre boyunca saklanır. 4. Verilerin Paylaşımı Toplanan bilgiler hiçbir koşulda üçüncü şahıslarla pazarlama amacıyla paylaşılmaz. Yalnızca yasal zorunluluk durumlarında (mahkeme kararı vb.) yetkili kurumlarla paylaşılabilir. 5. Veri Güvenliği Firebase güvenlik kuralları (Security Rules) ile yetkisiz erişim engellenir. API bilgileri yalnızca kullanıcının yetkilendirdiği işlemler için kullanılır. 6. Kullanıcı Hakları Kullanıcılar, hesaplarının ve kişisel verilerinin silinmesini talep edebilir. Bu talepler için hizlisoftyazilim@gmail.com üzerinden bizimle iletişime geçebilirsiniz. 7. Politika Değişiklikleri Bu gizlilik politikası zaman zaman güncellenebilir. Güncellemeler bu sayfada yayınlandığı andan itibaren geçerli olur.
